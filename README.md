## Homework

1. Создайте `CMakeList.txt` для библиотеки *banking*.

    * Покрытие кода должно составлять 100%.
3. Настройте сборочную процедуру на **TravisCI**.
4. Настройте [Coveralls.io](https://coveralls.io/).
